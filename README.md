# Landing Page

A landing page built as part of **The Odin Project Foundations** curriculum. This project was created to practice building layouts with HTML and CSS, with a focus on Flexbox.

## Features

- Responsive page layout
- Navigation bar
- Hero section
- Information cards
- Quote section
- Call-to-action section
- Footer

## Built With

- HTML5
- CSS3
- Flexbox
- Git
- GitHub

## What I Learned

This project helped me become more comfortable with:

- Structuring webpages using semantic HTML
- Creating layouts with Flexbox
- Working with spacing, alignment, and positioning
- Organizing CSS for readability
- Using Git to track project progress

## Future Improvements

- Improve responsiveness for smaller screens
- Add subtle animations and hover effects
- Continue refining the design as I learn more CSS
